# RealAIze Capital - Landing Page

Landing page corporativa para RealAIze Capital, vehículo de inversión inmobiliaria con tecnología AI.

## Tech Stack

- **Next.js 15** - React framework con static export
- **Tailwind CSS** - Utility-first CSS
- **Motion (Framer Motion)** - Animaciones
- **Lucide React** - Iconos

## Desarrollo

```bash
npm install
npm run dev
```

Abre [http://localhost:3001](http://localhost:3001)

## Build para producción

```bash
npm run build
```

Genera archivos estáticos en `/out` listos para hosting.

## Despliegue

Los archivos de `/out` se pueden subir directamente a cualquier hosting estático (Hostinger, Netlify, Vercel, etc.)
